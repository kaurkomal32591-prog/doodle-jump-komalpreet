# Doodle Jump Game (p5.js)

This is a Doodle Jump style game created using p5.js and the HTML canvas.

## Features
- Player movement (left/right)
- Gravity and jumping
- Platforms with collision
- Endless scrolling level
- Score counter
- Game over and restart
- Object-oriented programming (Player & Platform classes)

## Controls
- Left Arrow / Right Arrow: Move
- R: Restart after game over

## Technologies
- JavaScript
- p5.js
- HTML Canvas
